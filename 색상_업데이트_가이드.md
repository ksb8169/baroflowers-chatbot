## 🚀 배포 방법

### 1. 로컬 테스트
```bash
# 1. 폴더로 이동
cd C:\Users\오늘\Desktop\chatbot-webapp\baroflowers-v3-latest

# 2. 브라우저에서 열기
index.html 더블클릭

# 3. 색상 확인

```

### 2. GitHub 업로드
```bash

git add style.css

# 커밋
git commit -m "UI 색상 업데이트 - 브랜드 아이덴티티 적용"

# 푸시
git push origin main
```
