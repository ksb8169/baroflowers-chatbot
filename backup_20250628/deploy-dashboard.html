GitHub 업로드 실행

아래 명령어를 실행하여 GitHub에 업로드합니다:

cd C:\Users\오늘\Desktop\chatbot-webapp\baroflowers-v3-latest

또는 Git 명령어를 직접 실행:

git add .
git commit -m "feat: 통합형 UI 버전 추가"
git push origin main
